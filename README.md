# Bulletsh
![img](img/bulletsh.svg)

Bsh is a shell wrapper that sends push notifications using pushbullet.

## Prerequisites
- python 3.7+

## Installing Depenencies
```bash
pip install -r requirements.txt
```

## Running using python
```bash
python -m bulletsh
```
